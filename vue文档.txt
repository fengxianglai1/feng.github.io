一：构建vue项目
1.node安装

//淘宝镜像安装
2.npm install -g cnpm --registry=https://registry.npm.taobao.org 

vue-cli安装
3. cnpm install -g vue-cli

webpack安装

4.vue init webpack myweb --项目构建
项目名称
项目描叙
项目创建者
cd myweb   --进入项目
cnpm install  --安装依赖项
cnpm run dev  --启动工程，打开网站

5.安装element-ui
    1.cnpm install element-ui --save
    2.修改webpack.base.conf.js的配置 增加
     {
        test: /\\\\\\\\.css$/,
        loader: "style!css"
      },
     {
        test: /\\\\\\\\.(eot|woff|woff2|ttf)([\\\\\\\\?]?.*)$/,
        loader: "file"
      } 
    3.main.js导入elementui
      import ElementUi from 'element-ui'
      Vue.use(ElementUi)
    4.index.html中导入elementui css js 样式
	<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
   
	<script src="https://unpkg.com/element-ui/lib/index.js"></script>

6.sass 安装
